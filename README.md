# Phyterminal
Phyterminal is a 2D-physics renderer for <b>terminal</b>, which uses [Pymunk](https://github.com/viblo/pymunk) as its <b>physics engine</b>. Currently work in progress, but could be used in Windows, MacOS and Linux.
